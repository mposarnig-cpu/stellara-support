Häufige Fragen

  • Wie ändere ich mein Profil?
    Tippe in der App unten auf das Einstellungs-Symbol und wähle „Profil bearbeiten". Änderungen werden ab dem nächsten Tag berücksichtigt.

  • Wie kündige ich mein Abo?
    Abos können jederzeit über die iPhone-Einstellungen → Apple ID → Abonnements gekündigt werden.

  • Mein Kauf wurde nicht erkannt.
    Tippe in den Einstellungen auf „Käufe wiederherstellen". Falls das Problem weiterhin besteht, kontaktiere uns per E-Mail.

  • Wie werden meine Daten verwendet?
    Stellara speichert dein Profil ausschließlich lokal auf deinem Gerät. Es werden keine persönlichen Daten an Dritte weitergegeben.

Datenschutz

Stellara erfasst keine personenbezogenen Daten und gibt keine Nutzerdaten an Dritte weiter. Die App verwendet ausschließlich anonymisierte Anfragen an KI-Dienste zur Generierung von Horoskopinhalten. Dein Profil (Name, Geburtsdatum, Branche) wird nur lokal auf deinem Gerät gespeichert.
